# 🚀 Real-Time Object Detection Using Deep Learning (YOLOv8)

## 📌 Project Description
This project implements a real-time object detection system using YOLOv8 and OpenCV.  
It detects objects from images, videos, and live webcam streams.

---

## 🛠️ Technologies Used
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- NumPy

---

## 🎯 Features
- Detects 80+ object classes (COCO dataset)
- Supports image, video, and webcam input
- Displays bounding boxes with labels and confidence scores
- Confidence threshold: 0.6

---

## ⚙️ Setup

### Install dependencies
pip install -r requirements.txt

### Download model
Place yolov8x.pt in project folder

---

## ▶️ Run
python main.py
