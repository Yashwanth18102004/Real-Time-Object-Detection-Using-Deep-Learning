# 🚀 Real-Time Object Detection using YOLOv8

## 📌 Overview
This project implements a **real-time object detection system** using YOLOv8 and OpenCV.  
It is capable of detecting multiple objects from images, videos, and live webcam streams with high accuracy and speed.

---

## 🎯 Key Highlights
- 🔍 Detects **80+ object classes** (COCO dataset)
- ⚡ Real-time detection using **YOLOv8**
- 🎥 Supports **Image, Video, and Webcam input**
- 🧠 Uses **Deep Learning (Computer Vision)**
- 📦 Clean and modular **Python class-based implementation**

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Libraries:** OpenCV, NumPy  
- **Model:** YOLOv8 (Ultralytics)  
- **Tools:** Jupyter Notebook / VS Code  

---

## ⚙️ How It Works
1. Load YOLOv8 pre-trained model (`yolov8x.pt`)
2. Capture input (image / video / webcam)
3. Perform object detection using deep learning
4. Draw bounding boxes with labels & confidence scores
5. Display output in real-time
